This design is built with one SPIKE Prime Core Set (45678) and four frictionless pegs available in the SPIKE Prime Expansion Set. It is designed to be similar to WeDO Robust Structures lesson.
<br>
<br>
Step 1: Build the Earthquake Simulator<br>
Step 2: Create different buildings using available LEGO brick. Change the height, create wider and narrower bases, etc.<br>
Step 3: Create a program that varies the shaking of the platform. <br>
Step 4: Record which building topples first
